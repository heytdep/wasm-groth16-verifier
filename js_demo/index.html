Groth16 ZK-Snark Verification Algorithm

by 0x74 (tdep)

This is a demo of using my wasm-groth16-verifier crate to verify groth16 ZK-SNARKs (BLS12_377 elliptic curve) on your browser (in this example) or in any WASM vm given the crate's compatibility to WebAssembly.

The proof is verified by providing the proof, verifying key, and public inputs to a JS function that "binds" to a Rust one thanks to WASM.

The above function call returns a true boolean since the proof is valid, in my case, I created both the proof, circuit and verifying key with Zokrates.

If you open the console, and try verifying this same proof but with "3" instead of "4" as public input, you will see that the function returns false since the proof is incorrect (the circuit tries to prove that we know a private value n so that n.pow(2) == public_input, and the private value in our case is "2", so "3" as public input won't work).
